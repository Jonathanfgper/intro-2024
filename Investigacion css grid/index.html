Investigacion CCS

En CSS Grid, defines un contenedor como un grid container y luego defines las áreas dentro de ese contenedor. Puedes establecer el tamaño y la disposición de las filas y columnas, y luego colocar elementos HTML dentro de estas áreas definidas. CSS Grid proporciona un control preciso sobre la disposición de los elementos en la página.
